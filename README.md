# 🚀 Server Health Dashboard

A full-stack system monitoring dashboard built using Linux, Flask, and Machine Learning.

---

## 🔥 Features

### 🖥️ CLI Dashboard (Linux)
- Real-time CPU, Memory, Disk monitoring
- Process tracking
- Network information
- Logging system
- Interactive terminal UI

### 🌐 Web Dashboard
- Built using Flask
- Real-time system stats
- Interactive charts (Chart.js)
- Auto-refresh UI

### 🤖 AI Anomaly Detection
- Uses Isolation Forest algorithm
- Detects abnormal system behavior
- Displays anomaly status in UI

---

## 🛠 Tech Stack

- Linux (Bash scripting)
- Python (Flask)
- psutil
- scikit-learn
- HTML, CSS, JavaScript (Chart.js)
